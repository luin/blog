从基础到上架：使用 Swift 开发 iOS 应用

热身周 第三天

学习了两天的 Swift，是时候做一款小应用来秀秀身手了！在热身周的后三天里，我们将一步步地动手打造第一个 iOS 应用，实现一个单位转换小应用。

0. 课程目标

  1. 了解 Xcode 6 的集成开发环境结构
  2. 如何在 Storyboard 中拖拽控件
  3. 如何在模拟器中运行一个应用
  4. 如何在 Xcode 中使用 Git

1. 新建项目

开发一款 iOS 应用，绝大部分时间都需要和 Xcode 打交道。而创建一个 Xcode 项目则是开发 iOS 应用要做的第一件事。一个 Xcode 项目包含项目描述（如支持的设备、开发者信息等）、代码、界面视图和图片等资源。

首先启动 Xcode 6，新建一个 Single View Application:

点击下一步，在下面的界面中选择 Language 为 Swift。其它字段的含义为：

  1. Product Name：项目名称
  2. Organization Name：组织名称，可以是公司名或是自己的名字。这个信息不会影响上架
  3. Organization Identifier：用于创建应用 bundle 的唯一标示，推荐使用域名的倒序形式。比如我的域名是 zihua.li，这里就是 li.zihua
  4. Devices：指定你的项目的目标设备，如 iPhone、iPad 还是 Universal（同时支持 iPhone 和 iPad），在本次课程中都会选择 iPhone

接下来会要求选择文件存放的位置，选择好后点击 Create 即可。

2. 【视频】是时候 Hello World 了！

设计界面是开发一款 iOS 应用最常也最容易下手的地方。在 Xcode 中设计界面有 3 种方式，分别是 Storyboard、nib 和代码编程实现。前两个方式都是通过拖拽控件来生成界面，而后者则是使用代码创建控件并实现布局。这 3 种方式中 Storyboard 是最新的技术，也是本次教程主要使用的方式。不过在教程的后面也会介绍如何通过编程实现界面以及如何和 Storyboard 相整合。
